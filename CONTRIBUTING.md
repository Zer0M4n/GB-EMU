Thank you for your interest in contributing to **GB-EMU**.
This is a **Game Boy (DMG) emulator** written from scratch in **C++17** and compiled to **WebAssembly (WASM)** using **Emscripten**.

GB-EMU is an **educational emulation project**, where accuracy and documentation come first.

## General Rules
- Emulator core contributions are welcome (CPU, MMU, PPU, timers, interrupts)
- Bug fixes, optimization, and documentation
- Open an Issue before major changes

## Emulation Project
- Accuracy > hacks
- Hardware behavior must be justified (Pan Docs, real hardware, test ROMs)
- Deviations must be documented

## Artificial Intelligence (AI)
AI usage is allowed **with full disclosure**.
You must include:
- Exact prompt
- AI model used
- Which parts were AI-generated

Undeclared AI-generated contributions will be rejected.
 
## Code of Conduct
Be respectful, inclusive, and professional.

## Pull Requests
- Must build successfully
- Must be documented
- Must explain technical decisions
